
# Pritam Priyabrata Palai - Portfolio

A futuristic 3D portfolio website built with Next.js, Three.js, and Framer Motion.

## Features

- **Immersive 3D Experience**: Interactive 3D scene powered by `@react-three/fiber` and `@react-three/drei`.
- **Smooth Animations**: Seamless page transitions and scroll animations with `framer-motion`.
- **Responsive Design**: Mobile-friendly layout using Tailwind CSS.
- **Cosmic Theme**: Dark mode design with modern aesthetic.

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **3D Graphics**: Three.js, React Three Fiber
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
